# mysite

Portfolio website for GitHub Pages.

## Quick deploy to GitHub Pages

1. Push this repository to GitHub.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Set branch to **main** and folder to **/(root)**.
5. Save and wait for deployment to complete.

### Access URL
- Personal site repo (`<username>.github.io`): `https://<username>.github.io`
- Project repo (`mysite`): `https://<username>.github.io/mysite`

## Local preview

Run a simple static server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
